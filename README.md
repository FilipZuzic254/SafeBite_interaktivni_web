# SafeBite web aplikacija

Pri inicijalizaciji frontend-a i backend-a potrebno je preuzeti sve potrebne module

## Frontend

```
npm install
quasar dev
```

## Backend

```
npm install
node api.js / nodemon api.js
```

```
Izgled se može uzeti iz RegistracijeKorisnika ako hoćete da bude slično
```
